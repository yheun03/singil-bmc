AppInput

Text 입력을 위한 기본 Input 컴포넌트입니다. size / shape / icon / state / disabled / readonly 상태를 지원합니다.

Basic

가장 기본적인 Input 사용 예시입니다.

이름은 2자 이상 입력하세요

Size

Input의 높이를 제어합니다.

Shape

Input의 border 스타일을 제어합니다.

Icon

Input 내부에 아이콘을 배치할 수 있습니다.

Iconify

@iconify/vue 아이콘을 Input의 좌/우 슬롯에 배치하는 예제입니다.

State
