AppButton Demo

AppButton 컴포넌트의 모든 속성을 한 화면에서 확인합니다.

Variant

버튼의 기본 스타일을 정의합니다.

  • fill : 배경 채움 버튼
  • outline : 테두리 버튼
  • text : 배경 없는 텍스트 버튼
  • underline : 링크 스타일 버튼

Size

버튼 크기를 정의합니다.

  • xs
  • sm
  • md (default)
  • lg
  • xl

fill 버튼과 함께 사용

text / underline 버튼은 typography가 별도로 적용됩니다.

Shape

버튼 모서리 스타일

  • square : 직각
  • round : 기본 (6px)
  • pill : 완전 둥근 버튼

Tone

버튼 색상 스타일

  • primary
  • secondary
  • gray
  • danger
  • warning
  • success
  • info

Icon

iconLeft / iconRight 슬롯을 통해 아이콘을 추가할 수 있습니다.

Link

AppButton은 버튼 / 링크 / NuxtLink를 자동으로 처리합니다.

  • to → NuxtLink
  • href → a 태그
  • newTab → target="_blank"
내부 라우팅 외부 링크

State

버튼 상태

  • disabled : 클릭 불가
  • loading : spinner 표시
  • block : width 100%
  • custom-size : 사용자 정의 크기
