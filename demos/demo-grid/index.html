검색 폼 값 (v-model)

각 그리드 AppGridSearch에 연결된 반응형 검색 객체입니다. 필드를 조작하면 실시간으로 갱신됩니다. 

{
  "grid1": {
    "name": "",
    "verifiedPick": null,
    "qCol": "name",
    "qText": "",
    "grade": "",
    "departmentSet": [],
    "activeOnly": false,
    "joinedAt": null
  },
  "grid2": {
    "product": "",
    "status": "",
    "date": null,
    "shipDate": null
  },
  "grid3": {
    "product": "",
    "listedAt": null
  }
}
