Modal Demo

modal host + store 기반 모달 데모입니다. 개별 모달 컴포넌트를 직접 렌더링하지 않고, handler에서 modalOpen(...)만 호출해 Alert / Confirm / Custom 모달을 엽니다.

Basic

가장 기본적인 Alert / Confirm / Custom 모달 오픈 예시입니다.

Alert Props

title, message, confirmText, closeOnDim, closeOnEsc, closable 조합 예시입니다.

Confirm Props

confirmText, cancelText, closeOnDim, closeOnEsc, closable 조합 예시입니다.

Custom Modal

component와 componentProps를 전달해 원하는 내용을 렌더링하는 방식입니다. 실무에서는 별도 컴포넌트 전달 방식이 유지보수에 유리합니다.

Viewer Modal

이미지/PDF 미리보기처럼 `custom` 모달을 재사용하는 예시입니다.

Store Control

스토어 기준으로 최상단 모달 닫기, 전체 모달 닫기 같은 제어도 가능합니다.
